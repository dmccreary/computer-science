# Monty the Python - Mascot Guide

Monty is our friendly learning mascot who guides you through this
AP Computer Science textbook. Monty is a bright green python snake
who loves helping students learn to code!

## Meet Monty

**Name:** Monty the Python
**Species:** Python snake
**Personality:** Friendly, encouraging, curious, adventurous, and playful
**Catchphrases:** "Let's code this!", "Let's debug this together!", "You've got this!"

Monty wears a gold coding hoodie and headphones, and is always
ready to help you tackle the next programming challenge.

## Monty's Appearances

Monty shows up in six different ways throughout the textbook:

### Welcome

!!! mascot-welcome "Monty Says: Let's Code This!"

    Welcome to a new chapter! Monty appears at the start of
    each chapter to introduce what you'll learn and get you
    excited about the topics ahead.

### Key Insight

!!! mascot-thinking "Key Insight"

    When there's an important concept to understand, Monty
    takes a moment to think deeply and share the core idea
    with you.

### Tip

!!! mascot-tip "Monty's Tip"

    Monty shares helpful tips and shortcuts that make coding
    easier. These are practical nuggets of wisdom you'll use
    again and again.

### Warning

!!! mascot-warning "Watch Out!"

    When there's a common mistake or pitfall, Monty warns
    you before you fall into the trap. Pay attention to these!

### Celebration

!!! mascot-celebration "Great Progress!"

    When you've mastered a topic or completed a major section,
    Monty celebrates with you. You earned it!

### Encouragement

!!! mascot-encourage "You've Got This!"

    When the material gets tough, Monty is there to remind
    you that struggle is normal and you can do it.

## Usage in Chapters

Authors use standard admonition syntax with these custom types:

```markdown
!!! mascot-welcome "Monty Says: Let's Code This!"
    Welcome text here...

!!! mascot-thinking "Key Insight"
    Important concept explanation...

!!! mascot-tip "Monty's Tip"
    Helpful tip text...

!!! mascot-warning "Watch Out!"
    Warning about common mistake...

!!! mascot-celebration "Great Progress!"
    Celebration text...

!!! mascot-encourage "You've Got This!"
    Encouraging message...
```

## Placement Rules

| Context | Admonition Type | Frequency |
|---------|----------------|-----------|
| Chapter opening | mascot-welcome | Every chapter |
| Key concept | mascot-thinking | 2-3 per chapter |
| Helpful tip | mascot-tip | As needed |
| Common mistake | mascot-warning | As needed |
| Section completion | mascot-celebration | End of major sections |
| Difficult content | mascot-encourage | Where students struggle |

**Important:** Monty should appear no more than 5-6 times per chapter.
Don't place mascot admonitions back-to-back.
